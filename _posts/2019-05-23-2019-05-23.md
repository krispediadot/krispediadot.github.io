## Git 
#### 직전에 commit한 commit 메세지 수정하기

두가지 방법이 있음<br>

- `git commit --amend`<br>
    해당 위치에 가서 git commit --amend 하면 파일 수정 할 수 있음  

- `git commit --amend -m "수정하는 메세지"`<br>
    수정하는 메세지를 적으면 바로 수정할 수 있음
