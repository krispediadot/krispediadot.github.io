### 1. 설치
---
```
sudo apt-get install screen
```

### 2. 사용
---
`screen -S 세션이름`: 특정 세션이름으로 새로운 스크린 생성  
`screen -r 세션이름`: 특정 세션 attach  
`exit`: 해당 세션 제거
`Ctrl+a d`: 기본 스크린으로 이동
